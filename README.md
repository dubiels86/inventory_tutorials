# Tutoriales de Inventario

## Descripción
Este módulo proporciona tutoriales interactivos para aprender a trabajar con el módulo de inventario de Odoo.

## Características
- Tutoriales organizados por categorías (Básico, Operaciones, Reportes, Avanzado)
- Contenido HTML enriquecido
- Vista kanban para navegación fácil
- Secuenciación de tutoriales

## Instalación
1. Copia el módulo en tu directorio de addons
2. Actualiza la lista de aplicaciones
3. Instala el módulo "Tutoriales de Inventario"

## Uso
1. Ve al menú "Tutoriales Inventario"
2. Explora los tutoriales disponibles
3. Haz clic en cualquier tutorial para ver su contenido

## Tutoriales Incluidos
1. **Conceptos Básicos de Inventario** - Fundamentos del módulo
2. **Crear y Configurar Productos** - Gestión de productos
3. **Operaciones de Inventario** - Recepciones, entregas, transferencias
4. **Reportes de Inventario** - Análisis y reportes
5. **Configuración Avanzada** - Configuraciones especializadas

## Personalización
Puedes agregar nuevos tutoriales creando registros en el modelo `inventory.tutorial`.
